# **App Name**: Ferretools Inventory

## Core Features:

- Product Catalog: Display available items with names, images, descriptions, prices, and quantity. Out-of-stock items must have visual emphasis, like a badge or banner across the picture.
- Inventory View: Displays all items.
- Item count editor: Displays an input form to edit inventory count for each item.
- Add Item: Functionality to create/add new inventory items with relevant fields.

## Style Guidelines:

- Primary color: Strong blue (#4285F4), symbolizing reliability and trust.
- Background color: Light gray (#F0F2F5), creating a clean, neutral backdrop.
- Accent color: Muted orange (#FFA07A), drawing attention to interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif offering a modern, yet warm, user experience. It's suitable for both headings and body text, maintaining readability across different sections of the app.
- Use clear, minimalist icons representing product categories and actions (e.g., a hammer icon for 'tools,' a plus sign for 'add item').
- Employ a clean and intuitive layout with a sidebar menu for navigation and a card-based design for product display. Use white space effectively to avoid clutter.
